# Major Checklist

Use for:

- multi-file refactors
- migrations without irreversible data operations
- tool-heavy workflow changes
- integration changes

## Required

- explicit task brief
- dependency and contract review
- targeted verification plus one broader integration check
- adversarial pass completed
- rollback or containment path described
- external facts verified from primary sources

## For agent workflows

- tool surface narrowed
- approval boundary reviewed
- prompt injection paths considered
- commentary vs final separation considered
