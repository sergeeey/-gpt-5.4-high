# Evals

This directory holds the evaluation scaffolding for recurring GPT-5.4 workflows.

The purpose is not to create fake certainty.
It is to make prompt, tool, and model changes measurable.

Primary sources:

- https://developers.openai.com/api/docs/guides/model-selection
- https://developers.openai.com/api/docs/guides/prompt-optimizer
- https://developers.openai.com/api/docs/guides/agent-builder-safety

## Layout

- `failure-taxonomy.md` common failure categories
- `datasets/` sample synthetic datasets
- `graders/` rubric and grading guidance

## Rule

All datasets in this directory are synthetic examples unless explicitly stated otherwise.
